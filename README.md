#  Time-Series Stock Forecasting (AAPL & MSFT)
**ARIMA & Prophet Forecasting Models for Apple (AAPL) and Microsoft (MSFT)**  
A complete end-to-end time-series forecasting project built using Python, ARIMA, and Facebook Prophet.  
This project compares short-term vs long-term forecasting performance on two major stock tickers:
- **AAPL (Apple Inc.)**
- **MSFT (Microsoft Corp.)**

---

##  **Project Goals**
- Fetch historical stock prices using `yfinance`
- Perform time-series preprocessing & stationarity checks
- Build **ARIMA** model (good for short-term patterns)
- Build **Prophet** model (good for long-term trends & seasonality)
- Compare accuracy of both models
- Visualize forecast results
- Save trained models for reuse

---

##  **Technologies Used**
- **Python**
- **yfinance**
- **pandas**
- **numpy**
- **matplotlib**
- **statsmodels (ARIMA)**
- **prophet (Facebook Prophet)**
- **joblib** (for saving models)
